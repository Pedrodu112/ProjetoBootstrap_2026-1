CHECKPOINT — SITE COM HTML + CSS + BOOTSTRAP

1. BOOTSTRAP (CDN PARA USAR NO PROJETO)

Colocar dentro do <head> do HTML:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

Colocar antes de fechar o </body>:

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

3. BOTÕES BOOTSTRAP (SUBSTITUI .botao)

Exemplo:

<a href="cardapio.html" class="btn btn-danger mt-3">Ver Cardápio</a>

Classes comuns:

btn btn-primary   -> azul
btn btn-danger    -> vermelho
btn btn-success   -> verde
btn btn-warning   -> amarelo
btn btn-dark      -> preto

---

4. CLASSES IMPORTANTES DA NAVBAR

navbar
→ cria a barra de navegação

navbar-expand-lg
→ menu vira botão em telas pequenas

navbar-dark
→ texto claro para fundo escuro

container
→ centraliza o conteúdo

navbar-brand
→ nome ou logo do site

navbar-toggler
→ botão do menu no celular

navbar-toggler-icon
→ ícone de 3 linhas

collapse navbar-collapse
→ abre e fecha o menu no celular

navbar-nav
→ lista de links do menu

nav-item
→ cada item do menu

nav-link
→ estilo dos links

ms-auto
→ empurra menu para direita
